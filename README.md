# Setup
1. Install yarn packages
    ```bash
    yarn install
    ```
2. Start backend server
    ```bash
    cd backend
    node server.js
    ```
    The server should be running on port 5000.

3. Start React app in another terminal
    ```bash
    yarn start
    ```
    The webpage should be running on [http://localhost:3000](http://localhost:3000).
